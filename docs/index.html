1. JavaScript and Node.js

JavaScript is a dynamic, multi‚Äëparadigm language. Node.js runs JS outside the browser using V8, enabling server‚Äëside code and CLI tools.

2. 8 Data Types in JS

JavaScript has 8 types: 7 primitive + 1 reference (object).

  • Primitives: string, number, bigint, boolean, undefined, symbol, null
  • Reference: object (arrays, functions, maps, sets, dates, regex, etc.)
typeof 42;           // 'number'
typeof 1n;           // 'bigint'
typeof Symbol('id'); // 'symbol'
typeof null;         // 'object' (legacy quirk)

3. Variables and Scope

const and let are block‚Äëscoped. var is function‚Äëscoped and hoisted.

console.log(x); // undefined
var x = 10;

// console.log(y); // ReferenceError
let y = 5;

Constants reference the same object but internal values can mutate.

const cfg = { mode: 'dev' };
cfg.mode = 'prod'; // allowed

4. Objects

Objects store key‚Äëvalue pairs. Keys are strings or symbols.

const user = { name: 'Alex', age: 30 };
user.role = 'admin';

Object.keys(user);   // ['name','age','role']
Object.entries(user); // [['name','Alex'],['age',30],['role','admin']]

const clone = { ...user }; // shallow copy
const deep = structuredClone(user); // deep copy

const arr = [1,2,3,4];

6. Functions

Functions are first‚Äëclass objects. Can be declared, assigned, or passed around.

function greet(name){ return `Hi ${name}` }
const square = x => x * x;

function sum(...nums){ return nums.reduce((a,b)=>a+b,0) }

console.log(sum(1,2,3));

7. Classes

Classes are syntactic sugar over prototypes. Support inheritance and private fields.

class Animal {
  #id = Math.random();
  constructor(name){ this.name = name }
  speak(){ console.log(`${this.name} makes noise`) }
}
class Dog extends Animal{
  speak(){ console.log(`${this.name} barks üê∂`) }
}
const rex = new Dog('Rex');
rex.speak();

8. Prototypes

Objects link to prototypes; classes abstract this mechanism.

function Counter(){ this.value = 0 }
Counter.prototype.inc = function(){ this.value++; return this.value }
const c = new Counter();
c.inc();

9. CommonJS Modules

// math.js
function add(a,b){ return a+b }
module.exports = { add };

// app.js
const { add } = require('./math');
console.log(add(2,3));

Modules are cached after first load. Use delete require.cache[path] to clear.

10. Asynchronous JS & Event Loop

  try {

12. Tips & Best Practices

  • Prefer const over let; avoid var.
  • Use destructuring and template literals for clarity.
  • Handle promises with try/catch and always log errors.
  • Leverage built‚Äëins: Map, Set, Object.freeze, structuredClone.
